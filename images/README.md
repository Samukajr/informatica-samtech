# Pasta de Imagens

## Como Adicionar o Logotipo

Para adicionar o logotipo da empresa ao site:

1. **Renomeie seu arquivo de logotipo para**: `logo.png`
2. **Coloque o arquivo nesta pasta**: `images/logo.png`
3. **O logotipo aparecerá automaticamente no header do site**

### Especificações Recomendadas:
- **Formato**: PNG (com fundo transparente) ou JPG
- **Tamanho recomendado**: 200x200 pixels ou maior
- **Nome do arquivo**: `logo.png` (minúsculas)

### Como Adicionar pelo GitHub:
1. Acesse: https://github.com/Samukajr/informatica-samtech/tree/main/images
2. Clique em **"Add file"** → **"Upload files"**
3. Arraste seu arquivo `Logo.png` ou clique para selecionar
4. Renomeie para `logo.png` (minúsculas)
5. Clique em **"Commit changes"**

### Como Adicionar Localmente:
1. Copie seu arquivo `Logo.png` para esta pasta
2. Renomeie para `logo.png`
3. Faça commit: `git add images/logo.png`
4. Push: `git push`

**Nota**: O site já está preparado para usar o logotipo. Assim que você adicionar o arquivo `logo.png` nesta pasta, ele aparecerá no cabeçalho do site!
